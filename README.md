## 项目结构
- data/stock/index/macro/finance：原始数据（未上传，可通过代码重新下载）
- data/clean/：清洗后数据（已上传）
- data/combined/：合并后综合数据（已上传）

- 03_analysis.ipynb：描述统计与回归分析
- report.html：最终分析报告
- download_log.txt：下载日志
