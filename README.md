## 二、数据清洗过程（6个规范步骤）
 3.1. **缺失值处理**：删除价格、日期为空的异常行，保证时间序列连续
 3.2. **异常值剔除**：剔除涨跌幅超过±20%的异常交易数据
 3.3. **格式统一**：日期统一为`YYYY‑MM‑DD`，股票代码统一为6位字符串
 3.4. **类型转换**：价格、成交量强制转为数值型，避免字符串运算报错
 3.5. **去重处理**：删除重复交易日数据
 3.6. **筛选有效样本**：保留全样本周期内连续交易的个股数据

清洗后数据无空值、无异常值，可直接用于收益率计算与回归建模。

### 问题讨论（文字回答）
1. **β>1的股票、行业与周期性匹配性**
β＞1的股票集中在**能源、汽车制造、通信**等周期性行业；β＜1集中在银行、电力、交通运输等防御性行业，完全符合行业风险特征：周期行业受宏观经济影响大，系统性风险更高；防御行业需求稳定，风险更低。

2. **α是否显著及含义**
部分股票α的p值＜0.05，显著异于0。α显著为正代表对冲大盘风险后仍存在超额收益；α显著为负代表长期跑输市场；α不显著说明收益完全由市场风险解释。

3. **$R^2$差异解释**
$R^2$高的多为大盘权重股，收益主要受大盘系统性风险驱动；$R^2$低的多为中小盘个股，受公司特质风险、行业政策影响更大。

## 六、宏观指标（CPI）敏感性分析
模型：$r_{i,t}^{月}=\alpha_i+\gamma_i X_t+\varepsilon_{i,t}$

### 主要结论
1. 能源、汽车制造行业$\gamma>0$，CPI上涨带动企业盈利提升，收益上行
2. 银行、交通运输$\gamma<0$，通胀加息挤压行业利润，收益下行
3. 电力、物流行业对CPI无显著敏感性，行业定价受政策保护

整体符合宏观经济对不同行业的传导逻辑。

## 七、结论与投资启示
1. 行业风险分层清晰：周期股高β高波动，防御股低β低波动
2. 系统性风险是个股收益核心来源
3. 通胀对不同行业存在差异化影响
4. 可通过β系数与通胀敏感性构建行业轮动投资组合
